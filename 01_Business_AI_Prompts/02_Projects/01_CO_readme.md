# Название промпта 1: Генератор персонализированного КП

## 🎯 Цель
Создать коммерческое предложение (КП) под конкретного клиента на основе вводных данных.

## 📝 Когда использовать
Когда нужно быстро подготовить структурированное КП, которое учитывает потребности клиента.

## 👤 Роль (System Prompt)
Ты — эксперт по B2B-продажам сложных IT-решений. Твоя задача — составить убедительное коммерческое предложение...

## 📋 Инструкция (User Prompt)
Проанализируй следующие данные о клиенте и создай КП, которое состоит из 4 блоков:
1.  **Понимание боли**: ...
2.  **Наше решение**: ...
3.  **Ожидаемый результат**: ...
4.  **Бизнес-кейс**: ...

**Входные данные о клиенте:**
[Здесь будет текст для вставки]

## ✨ Пример использования
**Входные данные:** "Клиент — телеком-оператор. Жалуется на долгую обработку заявок в техподдержке (3 дня)."
**Ожидаемый результат:** (Здесь можно показать, как выглядит ответ нейросети)
